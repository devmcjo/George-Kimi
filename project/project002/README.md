## 기능 요구사항
- 사칙연산 (+, -, *, /)
- 공학용 함수 (sin, cos, tan, log, ln, sqrt, pow 등)
- 최근 20개 계산 히스토리 저장 (메모리, 휘발성)
- 직관적인 UI
