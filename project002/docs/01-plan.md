## 3. 검증 방법론

### 3.1 3회 반복 검토 (Three-Pass Review)

각 검증 항목에 대해 다음의 3단계 검토 수행:

```
┌─────────────────────────────────────────────────────┐
│  1차 검토: 개요 파악 및 주요 이슈 식별               │
│     ↓                                              │
│  2차 검토: 심층 분석 및 상세 이슈 도출               │
│     ↓                                              │
│  3차 검토: 교차 검증 및 누락 항목 확인               │
└─────────────────────────────────────────────────────┘
```

### 3.2 점수화 기준

| 평가 항목 | 배점 | 기준 |
|-----------|------|------|
| 코드 품질 | 25점 | 가독성, 유지보수성, 표준 준수 |
| 기능 완성도 | 25점 | 요구사항 충족, 버그 없음 |
| 보안성 | 25점 | 취약점 없음, 보안 모범사례 준수 |

| **총점** | **100점** | |
